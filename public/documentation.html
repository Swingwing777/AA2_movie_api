Get all movies listed by title only /movies/titles GET -- None – via URL only JSON : Single object containing list of movie names:
[ "Dr. No", "From Russia With Love", "Goldfinger", etc, etc, "Skyfall", "Spectre" ]
Get One Movie by Title /movies/title GET /title must match case and spelling of database None – via URL only JSON: Single object containing information on single movie, as per first example for ' Get All Bond Movies' .
Fault message returned if no title match.
Get genres of specified Title /movies/genres/title GET /title must match case and spelling of database None – via URL only Text response outlining assigned genres for selected movie:
Example: According to IMDB.com, Casino Royale (1967) has been assigned the following genre or genres: Comedy.
Fault message returned if no title match.

Add user favorite /users/username/title POST /username and /title must match case and spelling of database None – via URL only Text response detailing which movie has been added to favorites.
Duplicate favorites are prevented:
Example: haldw has added title "Octopussy" to his/her/their favorites collection.
Example: haldw has already added title "Dr. No" to his/her/their favorites collection. Duplicate not added.
Remove user favorite /users/username/title/td> DELETE /username and /title must match case and spelling of database None – via URL only Text response detailing which movie has been removed from favorites.
Missing favorites are flagged:
Example: haldw has removed "Spectre" from his/her/their favorites collection.
Example: haldw does not have title "Spectre" in his/her/their favorites collection.
Remove user by username /users/username DELETE /username must match case and spelling of database None – via URL only Text response detailing that user has been removed from the database.
Unidentified users are flagged:
Example: Username "Navy" has been deleted from the database.
Example: Username Navy does not exist. Please recheck Username to be deleted and try again.
