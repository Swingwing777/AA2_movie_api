Register new user 
/users
POST -- JSON request object in following format: 
{
 "Username": "Navy",
 "Password": "wetfeet",
 "Email": "scurvy@gmail.com",
 "Birthday": "1978-10-12"
}
JSON response object confirming information submitted,
with randomised ID and empty favourites array: 
{
 "_id": ObjectID("xxxxxxxxxxxx"),
 "Username": "Navy",
 "Password": "wetfeet",
 "Email": "scurvy@gmail.com",
 "Birthday": "1978-10-12",
 "FavoriteMovies": []
}

Remove user by username 
/users
/Username
 DELETE /username must match case and spelling of database None – via URL only

Text response detailing that user has been removed from the database.
Unidentified users are flagged. Examples:

Username "Navy" was deleted.


Username "Navy" was not found.
Remove user by ID 
/ident
/id
 DELETE /id must match id in database None – via URL only

Text response detailing that user has been removed from the database.
Unidentified users are flagged. Examples:

User with ID: 5f570121cef86914be0c8d87 was deleted.

User with ID: 5f570121cef86914be0c8d87 was not found.
